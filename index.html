⚡ Sats Giveaway Tag Writer

Paste a Lightning LNURL-withdraw link → get a QR + a Flipper .nfc file you can write to an NFC tag. 100% offline · nothing is sent anywhere · no wallet access.

Treat tags like cash. An LNURL-withdraw link is a bearer claim — anyone who taps or scans it can withdraw the sats. Use a single-use withdraw config on your backend so each tag can only be claimed once. This tool never touches your wallet; it only encodes a link you already created.

Create this on your own Lightning backend (e.g. a self-hosted LNbits → Withdraw Links). Paste the finished link here.

Tapping opens the recipient's Lightning wallet directly with the LNURL-withdraw claim.

Flipper NFC file

Writes an NDEF URI record onto a blank NTAG21x.

  1. Copy the .nfc file to /ext/nfc/ on your Flipper SD card.
  2. Open NFC → Saved, pick the file.
  3. Choose Write, then hold a blank tag to the Flipper.
  4. Tap the tag with a phone to test the claim before giving it away.

Single-file · client-side · no tracking. The decoded link stays in your browser.
